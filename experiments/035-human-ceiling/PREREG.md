# 035 — Human ceiling: PRE-REGISTRATION (locked before any annotation)

목적: 천장이 **데이터 부족**인지 **본질적 라벨 모호성**(사람도 이 입력만으론 못 푸는 confusion)인지
분리. 이것이 비싼 데이터 확장 전에 닫아야 할 마지막 질문이다.

## 고정 설계 (변경 금지)
- **입력 동등성:** 사람은 모델이 보는 것과 **픽셀 단위로 동일한 입력** — 전체 이미지를 518px로
  리사이즈한 뒤 핀을 마킹한 그림. tight crop 금지(모델은 crop을 안 본다; ViT 전역 attention).
- **메인 비교 = 215-way 객관식**(모델 closed-set과 정합). free-recall은 현실성 보조 지표.
- **블라인드:** 퀴즈 하니스는 정답·모델예측을 절대 표시하지 않는다. 채점은 별도 스크립트.
- **모델 예측:** 핀별 **specimen-LOO exemplar 1-NN**(같은 페이지 제외) → 사람과 같은 항목 비교.
- **표본:** n≈100, 혼동조직(동맥/정맥/신경) 가중 + 모델-정답/오답 균형(2×2 칸을 채우기 위함).
- **평가자:** (3) 지금은 생성자 본인 블라인드 파일럿. 패킷은 평가자-불문 → 나중에 의대생/전문가
  섭외되는 대로 채점만 추가. 평가자 유형은 메타에 기록.

## 1차 출력 = 오답 겹침 2×2 (숫자 한 개보다 결정적)
| | 모델 정답 | 모델 오답 |
|---|---|---|
| **사람 정답** | 둘 다 쉬움 | 🟢 **학습 가능 헤드룸** (데이터/방법 여지) |
| **사람 오답** | 모델 우위 | 🔴 **본질적 모호성** (데이터로 안 풀림) |

핵심 비율 + 95% bootstrap CI: **P(모델오답 ∩ 사람정답)** 와 **P(둘 다 오답)**.

## 해석 규칙 (결과 보고 끼워맞추기 방지 — 034 pre-reg와 동일 역할)
생성자 본인 = **오염된 상한**(표본을 기억할 수 있어 부풀려짐). 따라서:
- **본인 ≈50% (낮음):** 🔴 가장 강한 결론. *오염된 상한조차 낮다* → 진짜 값은 더 낮거나 비슷.
  외부 평가는 *확증용*이지 결론 전제가 아니다. + P(둘 다 오답) 높으면 "본질적 모호성" 확정.
- **본인 ≈85% (높음):** 🟡 기억 효과 배제 불가 → **외부 평가 필수**, 본인만으론 결론 불가.
- **본인 ≈65% (중간):** 🟡 외부 평가로 확정.

모델 기준선: 215-way top1 46.6 / top5 58 (canonical GaussianPool exemplar).
